# UAT Checklist: Story #8 — Watch Time Dashboard

Generated by /prd | Sources: story-8.md (acceptance criteria, user flows), story-8-ui-context.md (components, tokens)

---

## Setup

Platform: Web — open local dev server: `http://localhost:5173`
Test data:
  - Use account with at least one child profile already set up
  - Seed watch sessions via Firebase Emulator or use a test account with real session data:
    - At least one session today (e.g. 2700 seconds = 45 min)
    - At least one session earlier this week (different day)
    - At least one session earlier this month
  - For empty state test: use a fresh account with no watch sessions

Start at: Video Library screen (`/library`) — signed in, child profile set up, onboarding complete

---

## Checklist

### Flow 1: Parental Gate → Dashboard opens

- [ ] 1. Tap the lock icon (🔒) in the top-right corner of the Library screen
- [ ] 2. Verify: Parental Gate modal appears with a math question (e.g. "What is 3 + 8?")
- [ ] 3. Enter an incorrect answer — verify: question resets with a new question (no lockout)
- [ ] 4. Enter the correct answer
- [ ] 5. Verify: Gate dismisses, Parent Panel side drawer slides in from the right
- [ ] 6. Verify: Dashboard content is visible inside the panel

### Flow 2: Dashboard data display (with sessions seeded)

- [ ] 1. With sessions seeded: open the Parent Panel (solve gate)
- [ ] 2. Verify: "Today" label visible in grey text above the today value
- [ ] 3. Verify: Today's total shown in large purple text (e.g. "45 min" or "1 hr 30 min") — Baloo 2, ~28px
- [ ] 4. Verify: Weekly bar chart shows exactly 7 bars (Mon–Sun)
- [ ] 5. Verify: Today's bar is darker purple (#9333EA), other days are lighter (#C4B5FD)
- [ ] 6. Verify: Bar heights are proportional — the day with the most watch time has the tallest bar
- [ ] 7. Verify: All days have at least a small visible bar, even days with 0 seconds
- [ ] 8. Verify: Day labels "M T W T F S S" appear below the bars in grey
- [ ] 9. Verify: Monthly total appears below the chart (e.g. "This month: 2 hr 15 min")

### Flow 3: Empty state (fresh account, no sessions)

- [ ] 1. Switch to / sign in with a fresh account that has no watch sessions
- [ ] 2. Open Parent Panel (solve gate)
- [ ] 3. Verify: Empty state illustration/avatar and message visible: "No watch time recorded yet. Start a video to begin tracking."
- [ ] 4. Verify: No chart bars, no today value, no monthly total visible

### Flow 4: Panel dismiss

- [ ] 1. Open Parent Panel
- [ ] 2. Click the × close button (top-right of panel)
- [ ] 3. Verify: Panel slides out, Library screen is shown again
- [ ] 4. Click the overlay / area outside the panel (with another open)
- [ ] 5. Verify: Panel closes on overlay tap
- [ ] 6. Tap parent icon again — solve gate — verify panel reopens fresh

### Flow 5: Loading state

- [ ] 1. Open the Panel immediately after navigating to Library (slow connection or first load)
- [ ] 2. Verify: Shimmer skeleton bars visible while data loads (same shape as chart)
- [ ] 3. Verify: Shimmer is replaced by real data once loaded — no flash of empty state

### Flow 6: Time formatting edge cases

- [ ] 1. If you have a session of exactly 60 minutes (3600 seconds): verify "1 hr" (NOT "1 hr 0 min")
- [ ] 2. If you have a session of 90 minutes: verify "1 hr 30 min"
- [ ] 3. If you have only a very short session (< 60 seconds): verify "< 1 min"

---

## Visual Check

- [ ] Today value: Baloo 2 font, Extra Bold, ~28px, brand purple `#9333EA`
- [ ] "Today" label: Nunito, SemiBold, ~13px, grey `#6B7280`
- [ ] Monthly total: Nunito, SemiBold, ~15px, dark `#1E1B4B`
- [ ] Day labels: Nunito, ~11px, grey `#6B7280`
- [ ] Panel background: off-white `#FAFAFA`
- [ ] Panel left edge has rounded corners (24dp radius)
- [ ] Chart track bars: light purple `#E9D5FF`
- [ ] Dashboard feels calm and informative — no green/red colouring, no gamified score

---

## Responsive / Device Check

- [ ] At 480px width: panel fills most of the screen, content readable, no horizontal scroll
- [ ] At 768px width: panel width 320px, appears as side drawer on the right
- [ ] At < 480px: panel is full width, library visible behind overlay
- [ ] No horizontal scroll on Library screen at any tested width

---

## Edge Cases

- [ ] Child profile has ZERO sessions — empty state shown (not "0 min")
- [ ] Child profile has sessions but all of 0 seconds (e.g. video opened and immediately closed) — shows "0 min" or "< 1 min", NOT empty state
- [ ] Multiple sessions on the same day — all watchedSeconds correctly summed in today total
