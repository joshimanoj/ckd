# ckd — Choti Ki Duniya

Branded Android app for early childhood education (ages 6 months–6 years). A safe, distraction-free video platform that wraps YouTube content in a controlled, parent-friendly environment — inspired by the "Ms. Rachel" model of warm, curriculum-aligned early childhood video.

## Features

- **Safe Viewing Environment**: Distraction-free, branded interface — parents and children see Choti Ki Duniya, not YouTube
- **Curated Content**: Age-appropriate educational and entertainment videos for young children
- **Parent Controls**: Full control over what plays, with no algorithm-driven recommendations
- **Zero Hosting Cost**: Videos embedded from YouTube, presented in a branded wrapper
- **Firebase Backend**: User management, analytics, and content delivery

## Tech Stack

- React Native / Firebase
- YouTube embed integration
- Firestore (database)
- Firebase Auth

## Vision

Move the existing 6 lakh YouTube subscriber base onto a platform the creator controls — establishing direct family relationships, ownership of engagement data, and the foundation for a subscription business.
